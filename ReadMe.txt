***What you need***
-Matlab R2023b
-Computer Vision Toolbox addon

***Instructions***
-Double click to open main.m
-In function main() at the top of the code, there are two lines which call the the
different acquisition trajectories. Uncomment one line for which you wish to test that
trajectory (cartesian or radial).
-Parameters for each function: (image, number of lines, points per line). Change the image
as appropriate to test either of the test phantoms (IM1 or IM2) or the brain image (IM3).
-When done, press "Run".
